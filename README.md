pambu
=====

ncurses snake game.

     ENGLISH   │   MALAYALAM
    ───────────┼───────────────────
      snake    │     പാമ്പ്  (pambu)

REQUIREMENTS
------------

* `Python >= 3.4`

INSTALL
-------

    sudo make install

PLAY
----
Use arrow keys to move the snake. `<ESC>` to exit.
